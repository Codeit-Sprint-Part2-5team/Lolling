## 프로젝트 소개

  - Rolling은 롤링페이퍼를 웹에서 작성할 수 있는 서비스입니다.
  - 본인의 롤링페이퍼 페이지를 생성할 때 로컬에서 원하는 사진을 배경으로 등록해 보세요.
  - 다른 사람에게 롤링페이퍼를 작성할 때 프로필 사진도 로컬에서 등록할 수 있습니다.
  - 다른 사람의 롤링페이퍼에 이모지로 반응해 보세요.
